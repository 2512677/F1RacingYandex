If you want to use Realistic Engine Sounds 2 with Motorbike Physics Tool,
import Motorbike Physics Tool (make sure you done all the required steps to make MPT work), then import Realistic Engine Sounds 2 and later MPT_RES2-Pro.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and a prefab for Motorbike Physics Tool.
If you don't own Motorbike Physics Tool, you can buy it here: https://assetstore.unity.com/packages/tools/physics/motorbike-physics-tool-192691?aid=1101l3r5B
Read the documentation for tutorials.
