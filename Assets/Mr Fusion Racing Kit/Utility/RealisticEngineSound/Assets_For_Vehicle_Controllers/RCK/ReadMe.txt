If you want to use Realistic Engine Sounds 2 with Realistic Car Kit,
import Realistic Car Kit, Realistic Engine Sounds 2 and later RCK_RES2-Pro.unitypackage.
This unitypackage contain demo scenes, a script and a prefab made for Realistic Car Kit compatibility.
If you don't own Realistic Car Kit, you can get it here: https://assetstore.unity.com/packages/tools/physics/realistic-car-kit-18421?aid=1101l3r5B

Read the documentation for tutorials.
