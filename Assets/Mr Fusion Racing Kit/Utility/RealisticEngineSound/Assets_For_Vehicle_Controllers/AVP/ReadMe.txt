If you want to use Realistic Engine Sounds 2 with Arcade Vehicle Physics,
import Arcade Vehicle Physics (make sure you done all the required steps to make AVP work), then import Realistic Engine Sounds 2 and later AVP_RES2-Pro.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and a prefab for Arcade Vehicle Physics.
If you don't own Arcade Vehicle Physics, you can buy it here: https://assetstore.unity.com/packages/tools/physics/arcade-vehicle-physics-193251?aid=1101l3r5B
Read the documentation for tutorials.
