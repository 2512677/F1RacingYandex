import Sim-Cade Vehicle Physics (make sure you done all the required steps to make SCVP work), then import Realistic Engine Sounds 2 and later SCVP_RES2-Pro.unitypackage.
This unitypackage contain demo scenes, scripts for compatibility and a prefab for Sim-Cade Vehicle Physics.
If you don't own Sim-Cade Vehicle Physics, you can buy it here: https://assetstore.unity.com/packages/tools/physics/sim-cade-vehicle-physics-243624?aid=1101l3r5B
Read the documentation for tutorials.
